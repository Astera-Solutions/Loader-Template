  <p>
    This is a small Fabric loader template that grabs
    <a href="https://github.com/mioclient/oyvey-ported">OyVey-Ported</a>
    from a URL, keeps a cached copy, loads it during pre-launch, and then
    hooks up the metadata, mixins, access widener, and entrypoints so the
    remote mod behaves like it was already part of the client.
